== Description ==

Envosta is a unified FSE block theme designed as a parent theme for all Envosta-managed WordPress sites. It provides a semantic design token system, section block styles, and minimal template shells. Client customizations are applied via child themes and WXR content imports generated by the Envosta Studio.

== Features ==

* Semantic color palette (base, base-soft, contrast, accent, accent-hover)
* Section block styles (light, accent, dark) for alternating page sections
* Bundled variable fonts (Playfair Display, DM Sans)
* Minimal template shells — content built by the child theme and import
* GitHub auto-updater for seamless theme updates across all client sites
* Translatable template parts via hidden PHP patterns
